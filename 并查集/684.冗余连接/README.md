## 思路步骤

例子 [[1,2], [3,4], [3,2], [1,4], [1,5]]

🌟 边[a,b]意味着a所在集合可以和b所在集合合并。

1. 初始化容器，vector[i] = i; （第一个i是索引值，第二个i代表本节点的父节点）

    初始化 vector [0, 1, 2, 3, 4, 5]

    对应的下标值 [0, 1, 2, 3, 4, 5]
1. 遍历边集合，将边转化为集合的关系
    1. 针对[1, 2]
        1. vector[1] = 1, vector[2] = 2; (节点的父节点各不相同)
        2. 合并集合  vector[1] =  2 ; vector[2] = 2

             vector [0, 2, 2, 3, 4, 5]

            下标值 [0, 1, 2, 3, 4, 5]
    1. 以此类推....

[1, 4] 即最后一条闭合，即最后一条可以删除的边
